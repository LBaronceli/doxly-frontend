# Simple CRM Frontend

In combination with the backend, it allows the user to:
a) Register.
b) Log in.
c) Create client contacts with name, number, email, notes, and attachments.
d) Inspect the list of contacts.

Just run with

```bash
npm i
npm run dev
```
